# Demo Guide QM Cafe & Billiard

## Alur Demo Utama

1. Buka homepage `/` sebagai guest.
2. Lihat menu public di `/menu`.
3. Lihat paket billiard di `/billiard-packages`.
4. Lihat ketersediaan meja di `/table-availability`.
5. Login sebagai customer: `customer@qmcafe.com` / `Customer@12345`.
6. Customer buka `/customer/menu`, add menu ke cart, lalu checkout.
7. Customer buka order detail dan upload bukti pembayaran.
8. Login sebagai admin: `admin@qmcafe.com` / `Admin@12345`.
9. Admin cek users, menu, orders, bookings, payments, dan reports.
10. Admin approve/reject payment di Payment Verification Detail.
11. Login sebagai kitchen staff: `kitchen@qmcafe.com` / `Kitchen@12345`.
12. Kitchen staff memproses status order.
13. Login sebagai billiard staff: `billiard@qmcafe.com` / `Billiard@12345`.
14. Billiard staff melihat reservation, schedule, session, dan personal billing.
15. Login sebagai owner: `owner@qmcafe.com` / `Owner@12345`.
16. Owner melihat dashboard dan reports.
